### Плюсы реализации:
1. Простота реализации
2. Отсутствие внешних зависимостей
    1. Нет риска уязвимостей в сторонних библиотеках
    2. Меньший размер бандла
    3. Полный контроль над функциональностью drag&drop
3. Ограничение по границам при перетаскивании аннотаций
4. Визуальная обратная связь (opacity при перетаскивании, hover-эффекты)
5. Добавление аннотации по shift + click

### Минусы реализации:
1. Отсутствие бэкенда
    1. Состояние не сохраняется при перезагрузке страницы
    2. Просмотр единственной страницы
2. Проблема с производительностью при большом количестве аннотаций
    * *Решение - добавить ограничение на количество аннотаций*
3. При уменьшении масштаба (zoom) аннотации остаются большими и могут перекрывать большую часть текста
    * *Решение - добавить скейл для аннотаций*
4. Наложение аннотаций - аннотации могут перекрывать друг друга, сложно выделить нужную аннотацию
5. Отсутствие валидации текста аннотаций - пользователь может добавить слишком большой текст или оставить аннотацию пустой
    * *Решение - валлидировать текст аннотации при добавлении*
6. Нет адаптивной верстки под мобильные устройства
